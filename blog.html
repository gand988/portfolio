CSS3: video and some codes, easy.

CSS3: The box model. Box model is divided in two : block-level and inline-level.
Some common example about inline-levet tags are a, img, input, label. And is possible turning a block-level tag into a inline-level tag only modifing the CSS display: inline;. Block-level box takes an intire line.

The size of the block is calculated like this : content area width + padding-left/right + border-left/right + margin-left/right = full box width.

Padding is used to controll the size of a box without adjusting the size of the content inside the box.
h2 { padding: 0 0 0 0; }

Test

h2 { padding: 10px 0 10px 0; }

Test

Margin is used to controll the space between boxes. (invisible space)

Brower use a default sheet css to for tags. We can remove all of them (padding, border, margin) just including the tags that are in our HTML page and set to 0.

html, body, h1, h2, h3, p, ol, ul, li, a { padding: 0; border: 0; margin: 0;}
